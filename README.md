## Помощник при депрессии — Дневник настроения

Мобильное приложение для Android, помогающее отслеживать эмоциональное состояние и получать рекомендации по управлению настроением.

### Возможности

- **Журнал настроений** — записывайте своё настроение по 5-балльной шкале (от 😢 до 😊)
- **История записей** — просматривайте последние записи с датой и временем
- **Полезные советы** — 5 практических техник для улучшения эмоционального состояния
